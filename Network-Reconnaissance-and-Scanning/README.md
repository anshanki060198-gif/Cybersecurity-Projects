
# Reconnaissance and Scanning

## Project Overview
This project was completed as part of the IIT Kanpur Executive Program in Cybersecurity in collaboration with Simplilearn. The project focuses on reconnaissance and scanning techniques used in cybersecurity to gather information about target systems, identify network assets, and assess potential security vulnerabilities.

## Objectives
- Understand the role of reconnaissance in cybersecurity assessments.
- Perform passive and active information gathering.
- Discover network hosts and services.
- Identify open ports and running services.
- Analyze potential security weaknesses through scanning activities.

## Project Components

### Reconnaissance
- Open Source Intelligence (OSINT)
- Domain and DNS enumeration
- WHOIS lookup
- Search engine reconnaissance
- Target information gathering

### Scanning
- Host discovery
- Port scanning
- Service enumeration
- Network mapping
- Vulnerability identification

## Tools and Technologies
- Nmap
- WHOIS
- nslookup
- dig
- Traceroute
- Kali Linux
- Netdiscover

## Skills Gained
- Information Gathering
- Network Enumeration
- Security Assessment
- Vulnerability Identification
- OSINT Techniques
- Cybersecurity Fundamentals

## Learning Outcomes
- Gained hands-on experience with reconnaissance methodologies.
- Learned to identify network hosts and services.
- Developed skills in analyzing network infrastructure.
- Understood the importance of information gathering in security assessments.
- Applied scanning techniques in a controlled and authorized environment.
