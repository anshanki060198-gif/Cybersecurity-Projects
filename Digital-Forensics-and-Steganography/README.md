
# Digital Forensics and Steganography

## Project Overview
This project was completed as part of the IIT Kanpur Executive Program in Cybersecurity in collaboration with Simplilearn. The project focuses on the principles of Digital Forensics and Steganography, covering techniques for digital evidence analysis, data recovery, and hidden information detection.

## Objectives
- Understand the digital forensic investigation process.
- Collect, preserve, and analyze digital evidence.
- Study steganography techniques used to conceal information.
- Detect and extract hidden data from digital files.
- Apply cybersecurity best practices during forensic investigations.

## Project Components

### Digital Forensics
- Evidence acquisition and preservation
- File system analysis
- Metadata examination
- Log analysis
- Incident investigation fundamentals

### Steganography
- Information hiding techniques
- Image-based steganography
- Message extraction and analysis
- Detection of concealed data
- Security implications of steganographic methods

## Tools and Technologies
- Kali Linux
- Autopsy
- FTK Imager
- Wireshark
- Steghide
- OpenStego

## Skills Gained
- Digital Evidence Handling
- Forensic Investigation
- Data Recovery Techniques
- Steganography Analysis
- Cybersecurity Fundamentals
- Incident Response Concepts

## Learning Outcomes
- Developed an understanding of forensic investigation methodologies.
- Learned techniques for preserving and analyzing digital evidence.
- Explored methods of concealing and detecting hidden information.
- Gained practical exposure to industry-relevant cybersecurity tools.
