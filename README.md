# Enhancing Windows Security with Logon Monitoring and Reporting

## Project Overview
This project focuses on enhancing Windows security through logon monitoring and reporting. Windows Security Logs were analyzed to identify successful and failed logon attempts, detect suspicious activities, and improve visibility into user authentication events.

## Objectives
- Monitor user logon and logoff activities
- Detect failed login attempts
- Identify suspicious authentication behavior
- Generate security reports
- Strengthen Windows security monitoring

## Tools Used
- Windows Event Viewer
- Windows Security Logs
- PowerShell
- Splunk

## Skills Demonstrated
- Log Analysis
- Security Monitoring
- SIEM Fundamentals
- Incident Detection
- Windows Security Administration
